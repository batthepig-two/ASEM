  • Mutation Helper opens your breeding project tracker.
  • Stat Calculator opens the wild/fresh hatch stat point calculator and saved creature file list.
  • Tutorial opens this reference page.

Stat Calculator Buttons

  • Select Dinosaur: choose the species whose official base stats should be used.
  • Wild Dino: use this for untouched wild creatures.
  • Freshly Hatched Dino: use this for fresh babies or wild-stolen eggs with no imprint and no manual levels.
  • Baby/Creature Level: enter the level shown in game so ASEM can show visible points versus hidden speed points.
  • Stat value boxes: type the six visible in-game values: Health, Stamina, Oxygen, Food, Weight, and Melee.
  • − / + point buttons: nudge a stat down or up by exactly one wild/inherited point and fill the matching in-game value.
  • Save Creature: saves the selected dino, mode, level, stat values, calculated points, and chart data to the sidebar.
  • Saved Creatures list: click a saved item to reload that dino and redraw the six-stat chart; use Delete to remove it.

Mutation Helper Buttons

  • Add Another Breeding Project: creates a new saved project card.
  • Project name: click the title text to rename the breeding line.
  • ▼ / ▶: collapse or expand a project.
  • Target Stat: marks the stat you are currently trying to mutate.
  • Father/Mother stat levels: enter wild/inherited point levels, not displayed stat values.
  • Baby check boxes: enter a baby's stat levels to see whether it inherited the best stat or gained a new +2 mutation.
  • Mutation Stack − / +: track how many mutations are on the active side of the line.
  • Delete: removes a project from local storage.

Storage & Accuracy Notes

ASEM stores saved creatures and breeding projects only in your browser localStorage. Clearing browser data will remove them.

Fresh hatch calculations assume no imprint, no manual domestic levels, and default stat data from data.js. Wild-stolen egg creatures use the step-matching hatch logic built for Wyvern, Rock Drake, Magmasaur, and Deinonychus.
